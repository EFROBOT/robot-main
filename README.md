
## Installation

### Installer UV

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### Installer le projet

```bash
cd efrobot
uv sync
```

## Utilisation

### Lancer le programme

```bash
uv run start
```

### Commandes de développement

```bash
# Vérifier le code
uv run ruff check .

# Formater le code
uv run ruff format .
```
